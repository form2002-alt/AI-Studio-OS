## 1. 간단 요청 템플릿

아래 템플릿은 전시회사 AI 운영체제 v3.0에서 일러스트, 이미지 프롬프트, 실루엣, 지도, 생태 그래픽, 전환 공간 영상 콘셉트를 요청할 때 먼저 제시한다.

| 항목 | 입력 내용 |
|---|---|
| 프로젝트명 |  |
| 전시 유형 | 역사관 / 과학관 / 어린이체험관 / 기업홍보관 / 지역문화관 / 생태관 / 기타 |
| 이미지 용도 | 패널 / 사인 / 벽면 그래픽 / 활동지 / 카드뉴스 / 영상 콘셉트 / 아이콘 / 지도 |
| 주제·대상 |  |
| 시대·지역 |  |
| 관람 대상 | 일반 / 어린이 / 청소년 / 전문가 / 가족 관람객 |
| 원하는 화풍 | 수채화 / 실루엣 / 펜화 / 목탄 / 수묵 / 플랫 지도 / 생태 포스터 / 확인 필요 |
| 구성 | 전신 / 반신 / 장면형 / 지도형 / 상징형 / 벽면형 / 기타 |
| 정확성 조건 | 복식, 도구, 생물 형태, 지명, 문구 등 |
| 금지 요소 | 실사풍, 외국풍 혼입, 랜덤 문자, 과한 디테일 등 |
| 결과물 형식 | 이미지 생성 프롬프트 / 네거티브 프롬프트 / 검수표 / 수정 프롬프트 |

## 2. 기본 자동 적용 규칙

| 사용자 입력 | 자동 적용 |
|---|---|
| `수채화`, `부드러운 손그림`, `전시용 수채 인물` | 현대적 담백 수채화 + 얇은 라인아트 |
| `실루엣`, `아이콘`, `픽토그램` | 클린 블랙 실루엣·아이콘 스타일 |
| `1910년대`, `1920년대`, `일제강점기 한국 인물` | 흑백 펜앤잉크 + 약한 수채 워시, 한국 복식·신발 고증 우선 |
| `수탈`, `억압`, `상실`, `기억`, `저항` | 모노톤 목탄·파스텔 서사 / 흑백 입자 스포트라이트 / 미니멀 흑백 상징 컷 |
| `지역사 지도`, `탐방 지도` | 교육용 플랫 지도 일러스트 스타일 |
| `생태`, `해양`, `환경` | 컬러 플랫 생태 포스터 / 수채 벽화형 생태 전시 |
| `한국 자연`, `산하`, `프롤로그 배경` | 현대 수묵·수채 산수 / 청색 추상 수묵 산수 |

## 3. 출력 형식

| 순서 | 출력 내용 |
|---:|---|
| 1 | 요청 템플릿 확인 또는 누락 정보 표시 |

| 6 | 최종 이미지 생성 프롬프트 |
| 7 | 네거티브 프롬프트 |
| 8 | 생성 후 검수표 |
| 9 | 수정·재생성 프롬프트 |

## 4. 기본 네거티브 프롬프트

```text
실사풍 금지, 과도한 피부 묘사 금지, 지나친 주름과 모공 금지, 무작위 글자·로고·워터마크 금지, 복잡한 배경 금지, 자잘한 붓터치 과다 금지, 어둡고 과포화된 색 금지, 시대·복식 오류 금지, 일본식 복식·신발 혼입 금지, 요청하지 않은 중국풍 건축 금지, 손가락 오류 금지, 과장된 만화풍 금지
```

## 5. 검수 기준

| 검수 항목 | 질문 |
|---|---|
| 스타일 | 지정한 표준 화풍과 일치하는가? |
| 용도 | 패널, 사인, 벽면, 영상 등 실제 사용처에 맞는가? |
| 정확성 | 시대·복식·도구·생물·지형·문구가 사실과 맞는가? |
| 한국 맥락 | 일본풍·중국풍 요소가 의도 없이 섞이지 않았는가? |
| 가독성 | 원거리 또는 작은 크기에서도 읽히는가? |
| 제작 가능성 | 출력, 커팅, 벽면 시공, 애니메이션화가 가능한가? |
| 저작권 안전성 | 특정 작가·브랜드·작품 모방으로 보이지 않는가? |
