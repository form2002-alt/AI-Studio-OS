## 1. 반영 목적

이 문서는 `07_ILLUSTRATION_STUDIO/STYLE_SYSTEM_REFERENCE.md`와 `07_ILLUSTRATION_STUDIO/WATERCOLOR_STYLE_REFERENCE.md`의 일러스트 화풍 지침을 전시회사 AI 운영체제 v3.0에 연결 반영하기 위한 패치 문서이다.

| 목적 | 반영 내용 |
|---|---|
| 전시 일러스트 표준화 | 전시·교육·역사·생태·어린이 콘텐츠에서 사용할 표준 화풍을 통합 적용한다. |
| 엔진 간 일관성 | Illustration Engine, QA Engine, Render & Video Engine이 같은 스타일 기준을 사용한다. |
| 제작 오류 감소 | 과도한 실사화, 특정 작가풍 모방, 일본풍·중국풍 혼입, 랜덤 문자 오류를 줄인다. |
| 출력 안정성 | 패널, 사인, 벽면 그래픽, 영상 콘셉트, 활동지에 맞는 가독성과 제작 가능성을 확보한다. |

## 3. 전시회사 AI 운영체제 v3.0 공통 반영 규칙

| 항목 | 반영 규칙 |
|---|---|
| 템플릿 우선 | 일러스트·이미지·영상 콘셉트 요청이 들어오면 먼저 간단한 용도별 요청 템플릿을 제시한다. |
| 스타일명 표준화 | 모든 일러스트 결과물에는 표준 스타일명을 명시한다. |
| 특정 작가풍 금지 | 작가명, 브랜드명, 특정 작품명 모방 대신 선·색·질감·명암·구도 중심으로 스타일을 지시한다. |
| 수채화 기본값 | 사용자가 `수채화`, `부드러운 손그림`, `전시용 수채 인물`, `현대적인 수채화`라고만 요청하면 `현대적 담백 수채화 + 얇은 라인아트`를 기본 적용한다. |
| 실루엣 기본값 | 사용자가 `실루엣`, `아이콘`, `픽토그램`이라고 요청하면 `클린 블랙 실루엣·아이콘 스타일`을 기본 적용한다. |
| 한국 역사·문화 보호 | 한국 역사·문화 주제에서는 일본식 복식, 일본식 신발, 중국식 건축·문양, 무작위 외국어 문자를 금지한다. |
| 확인 필요 표시 | 시대, 지역, 복식, 도구, 동식물, 지명, 문구가 불확실하면 결과물에 넣지 않고 `확인 필요`로 표시한다. |
| 제작 가능성 | 출력 크기, 원거리 가독성, 선 굵기, 색 대비, 커팅·벽면 출력 가능성을 검토한다. |

## 5. QA Engine v3.0 반영 규칙

이미지 검수 시 기존 사실·안전·출력 검수에 아래 항목을 추가한다.

| 검수 항목 | 확인 기준 | 판정 |

| 표준 스타일 일치 | 결과물이 지정된 표준 스타일명과 일치하는가? | 적합 / 수정 / 확인 필요 |

| 실루엣 기준 | 실루엣 요청 시 작은 크기에서도 외곽 형태가 즉시 인식되는가? | 적합 / 수정 / 확인 필요 |
| 역사·문화 혼입 | 한국 주제에 일본풍·중국풍 복식, 신발, 건축, 문자가 섞이지 않았는가? | 적합 / 수정 / 확인 필요 |
| 문자 오류 | AI가 생성한 무작위 문자, 틀린 한글, 의미 없는 영문이 없는가? | 적합 / 수정 / 확인 필요 |
| 전시 가독성 | 패널·사인·벽면에서 원거리 가독성이 확보되는가? | 적합 / 수정 / 확인 필요 |
| 출력 제작성 | 인쇄, 컷팅, 벽면 출력, 애니메이션화가 가능한가? | 적합 / 수정 / 확인 필요 |
| 저작권 안전성 | 특정 작가·브랜드·작품을 직접 모방한 표현으로 보이지 않는가? | 적합 / 수정 / 확인 필요 |

## 7. 전시 일러스트 프롬프트 필수 항목

| 순서 | 항목 | 필수 내용 |
|---:|---|---|
| 1 | 프로젝트명 | 전시명 또는 콘텐츠명 |
| 2 | 전시 유형 | 역사관, 과학관, 어린이체험관, 기업홍보관 등 |
| 3 | 이미지 용도 | 패널, 사인, 벽면, 활동지, 카드뉴스, 영상 콘셉트 등 |
| 4 | 주제·대상 | 인물, 장면, 동물, 식물, 지도, 상징 이미지 등 |
| 5 | 시대·지역 | 역사·문화 주제는 필수. 불명확하면 `확인 필요` |
| 6 | 표준 스타일명 | 통합 화풍 20종 중 선택 |
| 7 | 선·색·질감 | 선 굵기, 색상 팔레트, 수채·목탄·점묘·벡터 등 |
| 8 | 구성 | 전신, 반신, 중앙 배치, 여백, 지도형, 벽면형 등 |
| 9 | 정확성 조건 | 복식, 도구, 생물 형태, 지형, 문자, 금지 요소 |
| 10 | 네거티브 프롬프트 | 실사풍, 과한 디테일, 외국풍 혼입, 무작위 텍스트 등 |

## 9. 운영 결론

전시회사 AI 운영체제 v3.0에서 일러스트·이미지·영상 콘셉트가 필요한 모든 작업은 이 패치 문서를 통해 `07_ILLUSTRATION_STUDIO/STYLE_SYSTEM_REFERENCE.md`를 공식 화풍 기준으로 참조한다. 세부 프롬프트 작성 시에는 반드시 스타일명, 정확성 조건, 금지 요소, 생성 후 검수 기준을 함께 제시한다.
