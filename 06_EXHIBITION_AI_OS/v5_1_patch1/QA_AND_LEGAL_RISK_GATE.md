# V5.1 QA·법적·계약 리스크 게이트

V5.1에서 QA는 AI의 최종 판정이 아니라 사용자·PM·클라이언트·제작처가 판단하기 위한 검토 보조 절차이다.

## QA 판정 주체

| 주체 | 역할 |
|---|---|
| ChatGPT | 위험 신호, 오류 가능성, 수정 제안 제공 |
| 사용자 / PM | 실무적 통과 여부 1차 판단 |
| 클라이언트 | 내용, 명칭, 정책, 승인 사항 최종 확인 |
| 제작처 | 출력·시공·송출 가능 여부 확인 |
| 최종 판정자 | 프로젝트별로 명시 |

## 기본 판정

| 판정 | 의미 | 조치 |
|---|---|---|
| 통과 | 바로 제작·납품 가능 | 최종 파일 패키징 |
| 수정 필요 | 오류가 명확함 | 수정 지시서 작성 |
| 확인 필요 | 정보·사양·출처가 부족함 | 클라이언트 또는 제작처 확인 |
| 제외 권장 | 제작·법적·품질 리스크가 큼 | 대체 경로 선택 |

## 법적·계약 리스크 체크

| 항목 | 확인 기준 | 판정 |
|---|---|---|
| AI 도구 약관 | 사용 도구의 상업 이용 가능 여부 | 사용 가능 / 확인 필요 / 사용 보류 / 제외 |
| AI 생성물 권리 | 저작권 인정 가능성, 소유권, 사용권 확인 |  |
| 유사성 리스크 | 특정 작가, 브랜드, 캐릭터, 사진과 과도한 유사성 확인 |  |
| 공공기관 납품 | AI 사용 고지 필요 여부 확인 |  |
| 계약서 | 클라이언트 계약서상 AI 사용 조항 확인 |  |
| 폰트 | 상업·공공 납품 사용 가능 여부 |  |
| 음원 | 전시·온라인·상업 사용권 확인 |  |
| 이미지·영상 소스 | 스톡, 공식자료, 생성물 라이선스 확인 |  |

## 제외 권장 이후 대체 경로

| 상황 | 대체 경로 |
|---|---|
| AI 이미지 품질 불량 | 재생성 → Photoshop 수작업 보정 → 스톡/공식 이미지 대체 |
| 고증·정보 오류 | 리서치 재확인 → 클라이언트 확인 → 문안 수정 |
| 영상 품질 불량 | 컷 축소 → 정지 이미지 모션화 → AE 수작업 전환 |
| 렌더링 비현실성 | Blender 단순 모델링 → Photoshop 보정 → 제안서용 이미지로 제한 |
| 저작권 리스크 | 사용 보류 → 라이선스 확보 → 공식자료·스톡 대체 |
| 일정 부족 | 범위 축소 → 우선순위 조정 → 일정·비용 재협의 |
