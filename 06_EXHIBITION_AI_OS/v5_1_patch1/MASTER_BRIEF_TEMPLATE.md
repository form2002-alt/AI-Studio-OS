# V5.1 Master Brief 템플릿

Master Brief는 프로젝트의 단일 기준 문서이다. AI가 작성한 Brief는 초안이며, 사용자가 확인·수정·승인한 버전만 확정 기준으로 사용한다.

```text

- 의뢰처:

- 작업 기간:
- 최종 납품일:

2. 분야 프리셋
- 적용 프리셋:
- 보조 프리셋:
- 적용 이유:

3. 핵심 목표
- 핵심 메시지:
- 관람객/사용자에게 남겨야 할 인상:
- 반드시 전달할 정보:

4. 대상 사용자
- 주요 대상:
- 보조 대상:

- 접근성 고려:

- 인쇄물:

10. 생성 도구·제작툴
- 기획·분석 AI:
- 이미지 생성 도구:
```

## Master Brief 확정 절차

| 단계 | 내용 |
|---:|---|
| 1 | AI가 Master Brief 초안을 작성한다. |
| 2 | 사용자가 목적, 범위, 산출물, 일정, 예산, 금지 요소를 확인한다. |
| 3 | 확인 필요 항목은 클라이언트·제작처에 질의한다. |
| 4 | 사용자가 승인한 버전만 확정 Master Brief가 된다. |
| 5 | 최종 QA는 확정 Master Brief 기준으로만 판정한다. |
