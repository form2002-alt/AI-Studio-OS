## 주요 확정 사항

| 항목 | 결과 |
|---|---|
| 시스템명 | 전시 시스템 v4.0 |
| 시스템 성격 | 범용 전시 AI 운영 시스템 |
| 적용 분야 | 역사, 문화, 예술, 과학, 생활, 현대, 자연, 어린이, 디지털, 홍보 |
| 기본 운영 | 짧은 요청 → 엔진 자동 조합 → 실무형 결과물 |
| 이미지 프롬프트 | 통합형 프롬프트 우선 |
| Style Archive | 승인 스타일 자동 저장·분류·재사용 |
| 인포그래픽 | 필수 기능으로 포함 |
| 영상 | 스토리보드, AI 영상 프롬프트, AE·Blender 연동 포함 |
| 렌더링 | 실사·3D 렌더링, 공간·집기·제품·미디어 설치 포함 |
| 제작 검토 | 1~3개월 단기 전시 기준, 예산·설치·납품 검토 포함 |
| QA | 내부 상세 검토, 사용자에게는 간략 결론형 |
