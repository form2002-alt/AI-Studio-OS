# 전시 시스템 v4.0

전시 시스템 v4.0은 전시 기획, 자료 조사, 문안 작성, 이미지·일러스트 제작, 인포그래픽, 영상 스토리보드, AI 영상 프롬프트, 실사·3D 렌더링, 제작·설치 검토, 최종 QA를 지원하는 범용 전시 AI 운영 시스템입니다.

## 적용 분야

- 역사
- 문화
- 예술
- 과학
- 생활
- 현대
- 자연
- 어린이 체험
- 디지털 전시
- 홍보 콘텐츠

특정 역사 전시나 특정 이미지 스타일에만 치우치지 않고, 범용 전시 실무에 적용하는 것을 목표로 합니다.

## 핵심 원칙

| 항목 | 기준 |
|---|---|
| 운영 방식 | 짧은 요청 → 필요한 엔진 자동 조합 → 실무형 결과물 제공 |
| 템플릿 | 작업 유형에 맞는 간단한 템플릿 우선 제시 |
| 사실 검증 | 확인 가능한 출처 기반, 불확실한 정보는 `확인 필요` 표시 |
| 이미지 프롬프트 | 공통/네거티브로 나누지 않고 통합형 프롬프트 제공 |
| Style Archive | 승인 스타일 자동 분석·분류·코드화·재사용 |
| 인포그래픽 | 필수 기능으로 포함 |
| 제작 검토 | 1~3개월 단기 전시 기준, 예산·설치·납품 검토 포함 |
| QA | 내부 검토는 상세하게, 사용자에게는 결론만 간략히 표시 |

## 기본 호출

사용자가 `호출`이라고 입력하면 다음 네 가지를 함께 보여줍니다.

1. 기본 사용 방식
2. 사용 가능한 엔진 목록
3. 대표 호출어
4. 작업 시작 예시

## 대표 호출어

- `이 내용 검증해줘`
- `패널 문안 써줘`
- `이미지 프롬프트 만들어줘`
- `이 스타일 저장해줘`
- `인포그래픽으로 정리해줘`
- `영상 스토리보드 만들어줘`
- `AI 영상 프롬프트 만들어줘`
- `렌더링 프롬프트 만들어줘`
- `제작 가능성 봐줘`
- `QA 해줘`
