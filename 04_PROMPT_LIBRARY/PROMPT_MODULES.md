# Prompt Modules

## Purpose Module

This output is created for `[use case]` and must be understandable for `[audience]`.

## Accuracy Module

The result must match the correct period, region, role, material, object, structure, and terminology.

## Exclusion Module

Avoid modern elements, unsupported details, random text, logos, watermarks, inaccurate costumes, and unbuildable structures.

## Output Module

Create a clear, production-ready result suitable for exhibition, education, content production, or client review.
