## 1. 시스템 업그레이드 목적

이 문서는 그동안 프로젝트에서 정리한 일러스트 화풍 대화를 통합하여, 전시·교육·홍보·AI 이미지 프롬프트 제작에 반복 적용할 수 있는 표준 화풍 시스템으로 정리한 것이다.

핵심 목적은 다음과 같다.

| 목적 | 내용 |
|---|---|
| 화풍 표준화 | 수채화, 수묵화, 펜화, 목탄, 실루엣, 플랫 그래픽, 지도, 생태 전시 그래픽을 용도별로 정리한다. |
| 제작 안정성 | AI 이미지 생성 시 결과가 과도하게 실사화되거나 특정 외국풍·작가풍으로 흐르는 오류를 줄인다. |
| 전시 적합성 | 전시 패널, 사인, 벽면 그래픽, 어린이 체험물, 영상 콘셉트, 카드뉴스에 맞는 스타일을 선택한다. |
| 프롬프트 일관성 | 스타일명, 선, 색, 질감, 명암, 배경, 금지 요소를 항상 함께 지시한다. |
| 검수 가능성 | 결과물을 스타일·정확성·가독성·제작 가능성 기준으로 평가한다. |

## 2. 공통 작동 원칙

| 구분 | 기준 |
|---|---|
| 템플릿 우선 | 사용자가 작업을 지시하면 먼저 해당 용도의 간단한 요청 템플릿을 제시한다. |
| 스타일 요소 중심 | 특정 작가명, 특정 브랜드, 특정 작품명 모방 대신 선·색·질감·구도·명암 등 시각 요소로 설명한다. |
| 사실 검증 | 시대, 복식, 동식물, 지역, 도구, 건축, 지명, 문구가 필요한 경우 확인되지 않은 내용은 `확인 필요`로 표시한다. |
| 전시 우선 | 미적 효과보다 교육적 이해, 원거리 가독성, 출력 안정성, 관람 동선 적합성을 우선한다. |
| 한국 맥락 보호 | 한국 역사·문화 주제에서는 일본풍·중국풍 복식, 신발, 건축, 문자 요소가 섞이지 않도록 명확히 금지한다. |
| 과잉 디테일 제한 | 주름, 모공, 속눈썹, 과도한 얼굴 묘사, 지나치게 작은 붓터치, 복잡한 배경을 기본적으로 줄인다. |
| 출력 안전성 | 작은 사인·아이콘은 단순형, 대형 벽면은 면·선 굵기·명암 대비를 확보한다. |

## 3. 최우선 수채화 기본값

사용자가 단순히 `수채화로`, `부드러운 손그림`, `전시용 수채 인물`, `현대적인 수채화`라고 요청하면 아래 스타일을 기본값으로 적용한다.

### 3-1. 기본 스타일명

**현대적 담백 수채화 + 얇은 라인아트**

### 3-2. 핵심 기준

| 요소 | 기준 |
|---|---|
| 선 | 얇고 절제된 펜선 또는 연필선. 굵고 검은 외곽선은 필요할 때만 사용한다. |
| 색 | 옅고 차분한 저채도 색. 따뜻한 베이지, 연한 황토, 부드러운 청회색, 연한 녹색, 자연스러운 피부톤. |
| 수채 표현 | 넓은 면의 투명한 워시, 부드러운 농담, 큰 명암 덩어리 중심. |
| 붓터치 | 자잘한 붓터치와 과도한 세부 붓질은 피한다. |
| 얼굴·손 | 최소한의 선과 색으로 표현한다. 실사풍 얼굴 묘사 금지. |
| 의상 | 옷 주름은 단순화하고 큰 명암 면으로 처리한다. |
| 배경 | 흰 배경 또는 매우 옅은 워시. 복잡한 배경은 주제상 필요할 때만 추가한다. |
| 질감 | 은은한 종이 질감과 흰 여백을 허용한다. |
| 분위기 | 조용하고 품격 있으며, 전시·교육 자료에 적합해야 한다. |

### 3-3. 금지 방향

- 어둡고 진한 고채도 색
- 오래된 느낌의 잔붓질 과다
- 과도한 주름, 모공, 수염, 피부 묘사
- 실사풍 초상화
- 복잡하고 지저분한 배경
- 무작위 텍스트, 로고, 워터마크

| 역사 인물 전신 삽화 | 01 또는 02 | 전시·교육용 가독성과 고증 설명에 적합 |

| 독립운동·수탈·억압·상실 | 03, 04, 05, 06 | 감정적·상징적 분위기와 전환 공간에 적합 |

## 6. 한국 역사·문화 주제 정확성 규칙

| 항목 | 지시 기준 |
|---|---|
| 복식 | 시대와 지역이 명시된 경우 해당 시기의 한국 복식을 우선한다. 모호하면 `확인 필요`로 표시한다. |
| 신발 | 한국 역사 인물에는 일본식 짚신, 게다, 조리, 와라지로 보일 수 있는 형태를 금지한다. 한국 짚신, 고무신 등은 주제와 시대에 맞을 때만 사용한다. |
| 인물 인상 | 한국인으로 보이도록 얼굴형, 헤어, 복식, 자세를 조정하되 고정관념적 과장은 피한다. |
| 배경 | 한국 농촌, 산하, 바다, 마을, 학교, 광산 등은 한국 맥락을 반영한다. 일본식 건축·문양·문자 혼입 금지. |
| 문자 | 한글·한자·영문 등 실제 표기가 필요하면 별도 검수한다. AI가 임의 문자 생성하지 않도록 한다. |
| 고증 불확실성 | 시대·복식·도구·장소가 불명확하면 결과물 안에 넣지 말고 요청서에 `확인 필요`로 표시한다. |

## 7. 프롬프트 작성 표준

AI 이미지 프롬프트는 아래 요소를 빠뜨리지 않는다.

| 순서 | 항목 | 작성 내용 |
|---:|---|---|
| 1 | 주제 | 무엇을 그리는지 명확히 쓴다. |
| 2 | 용도 | 전시 패널, 벽면, 사인, 활동지, 영상 콘셉트 등 용도를 쓴다. |
| 3 | 대상 | 인물, 동물, 식물, 풍경, 지도, 장면 등 대상을 쓴다. |
| 4 | 시대·지역 | 역사·문화 주제일 경우 반드시 명시한다. 모르면 `확인 필요`. |
| 5 | 스타일명 | 위 표준 스타일명 중 하나를 선택한다. |
| 6 | 선 | 얇은 선, 굵은 선, 펜선, 먹선, 실루엣 등 지시한다. |
| 7 | 색 | 흑백, 회청색, 저채도 수채, 민트톤, 블루 계열 등 팔레트를 지정한다. |
| 8 | 질감 | 종이 질감, 점묘, 목탄, 연필, 워시, 벡터 등 질감을 지정한다. |
| 9 | 구도 | 전신, 반신, 중앙 배치, 넓은 여백, 지도형, 벽면형 등을 지정한다. |
| 10 | 정확성 조건 | 복식, 도구, 동식물 형태, 지형, 문자 금지 등 검수 기준을 쓴다. |
| 11 | 네거티브 | 실사풍, 과한 디테일, 외국풍 혼입, 랜덤 텍스트, 로고 등을 금지한다. |

## 12. 향후 적용 지시

1. 사용자가 `수채화`라고만 말하면 **현대적 담백 수채화 + 얇은 라인아트**를 최우선 적용한다.
2. 사용자가 `실루엣`이라고만 말하면 **클린 블랙 실루엣·아이콘 스타일**을 적용한다.
3. 사용자가 역사 인물이나 1910~1940년대 한국 인물을 요청하면 복식·신발·시대 맥락을 먼저 점검한다.
4. 사용자가 스타일 참고 이미지를 제시하면 작가명 모방이 아니라 선·색·질감·명암·구도를 추출해 표준 스타일명으로 변환한다.
5. 어린이 전시에서는 어둡고 폭력적인 흑백 상징 스타일을 그대로 쓰지 말고 색감과 표현 수위를 완화한다.
6. 모든 프롬프트에는 스타일명, 정확성 조건, 금지 요소, 생성 후 검수 기준을 포함한다.
