# Watercolor Style Reference

## Use Case

Exhibition panels, public relations halls, museum character graphics, educational figures, and cultural storytelling illustrations.

## Visual Direction

A modern, calm watercolor figure style with thin line art and soft transparent washes.

## Key Features

| Element | Direction |
|---|---|
| Line | Thin, restrained ink or pencil-like outline. Avoid heavy black contours. |
| Color | Warm low-saturation colors. Use pale ochre, muted brown, soft blue-gray, beige, and gentle skin tones. |
| Wash | Broad transparent watercolor washes. Avoid excessive tiny brush marks. |
| Detail | Moderate detail only. Keep the figure readable from a distance. |
| Face and hands | Minimal line and color information. Avoid photorealistic facial detail. |
| Clothing | Use large value shapes and soft gradation. Keep folds simple. |
| Paper texture | Allow subtle paper grain and white gaps to remain visible. |
| Background | Mostly blank white or very light wash. Do not use complex scenery unless required. |
| Mood | Quiet, humane, educational, museum-appropriate. |

## Recommended Prompt Keywords

- modern watercolor figure illustration
- thin ink line art
- soft transparent wash
- muted warm colors
- broad watercolor areas
- subtle paper texture
- minimal facial detail
- clean white background
- museum educational illustration
- refined exhibition graphic

## Negative Direction

Avoid heavy black outlines, dark saturated colors, excessive wrinkles, photorealistic skin, over-detailed brushwork, vintage fine-art portrait style, messy background, random text, logos, and decorative clutter.

## Production Note

This style should be used as a preferred watercolor direction for exhibition and public-relations figure illustrations. It is suitable when the goal is clarity, dignity, warmth, and visual softness rather than dramatic realism.
