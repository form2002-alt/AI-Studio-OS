# 파일명 규칙

## 기본 형식

```text
YYYYMMDD_프로젝트명_작업유형_v001_작성자.md
```

## 버전 규칙

| 상태 | 표기 |
|---|---|
| 초안 | v001, v002, v003 |
| 검토본 | review_v001 |
| 최종 후보 | final_candidate_v001 |
| 최종본 | FINAL |
| 최종 수정본 | FINAL_REV01 |

## 금지 규칙

- `최종최종`, `진짜최종`, `최종2` 같은 파일명 사용 금지
- 날짜 없는 파일명 사용 금지
- 작업 유형을 알 수 없는 파일명 사용 금지
- 공백과 특수문자 남용 금지
